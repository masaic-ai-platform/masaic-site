# Masaic 



## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- Package manager: **Yarn**, **npm**, or **pnpm**

> This guide assumes you're using **Yarn**. You can adapt commands if you're using a different package manager.

---

### 🚀 Run Locally

#### 1. Install dependencies

```bash
yarn install

yarn build

yarn start


for Dev builds

yarn dev
